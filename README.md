# 📚 EduFlow — Smart Study Planner

EduFlow is a modern, responsive web application designed to help students **organize their study tasks, track progress, and stay focused** on academic goals.  
Built with **HTML, CSS, and JavaScript**, it provides a smooth and intuitive experience for planning your studies efficiently.

---

 
## 🚀 Features

- ✨ **Add and Manage Study Tasks**  
  Create, edit, and delete tasks with subject, due date, and priority levels.

- 🧠 **Smart Progress Tracking**  
  Displays real-time stats on total tasks, completed tasks, and overall progress with an animated progress bar.

- ⏰ **Reminders for Upcoming Tasks**  
  Automatic alerts for tasks due within the next hour to keep you on track.

- 🎨 **Beautiful UI/UX**  
  Elegant gradient design, animations, and responsive layout for all screen sizes.

- 💾 **Persistent Data with Local Storage**  
  Your tasks are saved automatically — even after you close or refresh the browser.

## ⚙️ Setup Instructions

1. **Clone this repository**
   ```bash
   git clone https://github.com/Umakantthosar07/EduFlow.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd EduFlow
   ```

3. **Open the app**
   - Simply open `Edu_Flow_index.html` in your web browser.

---

## 🧩 How to Use

1. Enter your **task title**, **subject**, **due date**, and **priority level**.  
2. Click **“Add Task”** to save it.  
3. Mark tasks as **Done ✅** or delete them 🗑️ anytime.  
4. Check your progress at the top — it updates automatically.
